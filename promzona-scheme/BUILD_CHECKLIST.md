## Перед билдом

- [ ] Запустить `npm install` для проверки зависимостей
- [ ] Запустить `npm run lint` для проверки кода
- [ ] Проверить все ссылки и изображения
- [ ] Тест всех зон (A, Б, С, Д)
- [ ] Тест модальных окон
- [ ] Тест переключения вкладок

## Сборка

```bash
# Установка зависимостей
npm install

# Линтинг
npm run lint

# Сборка
npm run build

# Предпросмотр
npm run preview
```
