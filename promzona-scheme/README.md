# Промзона Кара-Кече — Схема автоматизации

Интерактивная схема автоматизации угольной базы Кара-Кече (Кыргызкомур).

## 🚀 Быстрый старт

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```

### Сборка для продакшена
```bash
npm run build
```

### Предпросмотр билда
```bash
npm preview
```

## 📋 Структура проекта

- **Зона А** — Въезд, весы пустой тары, камеры ANPR/QR, выдача накладной
- **Зона Б** — 5 полос накопления, электронная очередь, распределение по категориям
- **Зона С** — Вызов транспорта оператором СБ, LED-экран, финальная проверка
- **Зона Д** — 3 весовых контейнера, взвешивание полной тары

## 🛠 Технологии

- React 19.2
- Vite 8.0
- Framer Motion
- React Router DOM

## 📦 Билд

После запуска `npm run build` готовые файлы будут в папке `dist/`.

Для деплоя скопируйте содержимое `dist/` на ваш сервер.
